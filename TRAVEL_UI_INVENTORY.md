## Existing state coverage

- Strong: Trips has skeleton, query error/retry, several context-specific empty states, confirmation, and export errors.
- Partial: dashboard has loading skeletons but no visible trips-query recovery; command palette has empty only.
- Partial: New Trip has inline validation, loading, draft restore, and toast feedback, but no dedicated submit error summary.
- Not assessed visually: English, Arabic, Hebrew, dark/light, browser, Electron, or viewport rendering. This is source inspection only.
